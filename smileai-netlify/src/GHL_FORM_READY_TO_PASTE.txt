Setup Instructions:

🚀 Quick Setup (5 Minutes)

STEP 1: Get Your GHL Credentials
• API Key: GoHighLevel → Settings → API → Create API Key
• Location ID: GoHighLevel → Settings → Business Profile → Location ID

STEP 2: Configure the App
• Click "Staff Login" in the footer
• Go to Settings → Integration tab
• Enter your API Key and Location ID
• Click "Save Settings"

STEP 3: Create Custom Fields in GHL
Navigate to: Settings → Custom Fields → Contact Fields

Create these 5 fields:

1. service_interest (Dropdown type)
   Options: Veneers, Invisalign, Whitening, Implants, General Checkup

2. transformation_status (Text type)
   Stores: Pending, Processing, Complete, Video Generated

3. before_image_url (Text or URL type)
   Stores: Original smile photo URL

4. after_image_url (Text or URL type)
   Stores: AI-enhanced smile photo URL

5. smile_video_url (Text or URL type)
   Stores: Animated smile video URL ⭐ IMPORTANT!

STEP 4: Test It!
✅ Submit a test lead on your landing page
✅ Check that the contact appears in GHL
✅ Upload a test photo and generate transformation
✅ Generate a test video
✅ Verify the video URL appears in the custom field and notes

🎥 Accessing Videos:
• Option 1: Open contact → Custom Fields → smile_video_url
• Option 2: Open contact → Notes → Look for "🎥 AI Smile Video Generated!"
