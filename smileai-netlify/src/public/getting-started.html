🦷 Welcome to SmileVisionPro AI!

Let's get you up and running in just 5 minutes

1
Get Your GHL Credentials

First, you'll need two pieces of information from GoHighLevel:

📌 API Key:

  • Go to GoHighLevel → Settings (⚙️)
  • Click "API" in the left sidebar
  • Click "Create API Key"
  • Name it: "SmileVisionPro AI"
  • Copy the API key (save it somewhere safe!)

📍 Location ID:

  • Go to Settings → Business Profile
  • Find "Location ID" at the top
  • Copy the Location ID
💡 Tip: Keep these credentials handy - you'll need them in the next step!
2
Configure SmileVisionPro AI

Now let's connect your app to GoHighLevel:

  • Open SmileVisionPro AI in a new tab
  • Scroll to the footer and click "Staff Login"
  • Click on "Settings" in the top navigation
  • Go to the "Integration" tab
  • Paste your API Key and Location ID
  • Click "Save Settings"
✅ Success! You should see "Settings saved successfully!" message.
3
Create Custom Fields in GHL

To store all the smile transformation data, create these 5 custom fields:

Navigate to: Settings → Custom Fields → Contact Fields → Add Field

Create these fields:

1. service_interest (Dropdown)
Options: Veneers, Invisalign, Whitening, Implants, General Checkup
2. transformation_status (Text)
Tracks: Pending, Processing, Complete
3. before_image_url (Text or URL)
Stores the original smile photo
4. after_image_url (Text or URL)
Stores the AI-enhanced smile photo
5. smile_video_url (Text or URL) ⭐
IMPORTANT: Stores the animated video link
⚠️ Important: Field names must match exactly (including underscores). Copy-paste them to avoid typos!
4
Test Your Setup

Let's make sure everything is working correctly:

  • Go to your SmileVisionPro AI landing page
  • Fill out the form with test information
  • Submit the form
  • Check GoHighLevel → Contacts
  • Verify a new contact was created
  • Upload a test photo and generate transformation
  • Generate a test video (optional)
  • Check that custom fields are populated
✅ Testing Complete! If you see the contact in GHL with all the data, you're all set!
5
Set Up Workflows (Optional)

Automate your follow-up with these recommended workflows:

🔔 New Lead Welcome

Trigger: Tag added "smile-transformation"
Actions: Send welcome email + SMS with booking link

🎥 Video Generated Follow-Up

Trigger: Custom field "smile_video_url" is not empty
Actions: Send email with video link + Add to "Hot Leads" pipeline
💡 Pro Tip: Contacts with videos convert 3x better! Set up automation to follow up within 10 minutes of video generation.
