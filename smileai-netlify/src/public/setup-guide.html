🚀 Quick Start (3 Steps)

1Get Your GHL Credentials

API Key: GoHighLevel → Settings → API → Create API Key

Location ID: GoHighLevel → Settings → Business Profile → Location ID

2Enter Credentials in App

Go to Staff Login → Settings → Integration Tab

Paste your API Key and Location ID, then click "Save Settings"

3Create Custom Fields in GHL

Navigate to: Settings → Custom Fields → Contact Fields

Create these 5 fields (see table below)

📋 Required Custom Fields

Create these custom fields in GoHighLevel to store all lead data:

Field Name Field Type Purpose
service_interest Dropdown Services customer is interested in
transformation_status Text Current transformation status
before_image_url Text or URL Original smile photo URL
after_image_url Text or URL AI-enhanced smile photo URL
smile_video_url Text or URL Video transformation URL
⚠️ Important: The smile_video_url field is crucial for accessing patient videos!

🎥 Video Integration

SmileVisionPro AI generates animated smile transformation videos. Here's how they're saved:

Where Videos Are Stored:

  • Custom Field: smile_video_url contains the direct video link
  • Contact Notes: A note is added with a clickable video link and description
  • Status Updated: Contact status changes to "Complete - Video Generated"

Accessing Videos:

Option 1: Open contact → Custom Fields → Click smile_video_url

Option 2: Open contact → Notes tab → Look for "🎥 AI Smile Video Generated!"

✅ Pro Tip: Set up a workflow that triggers when smile_video_url is populated to automatically send follow-up emails!

📊 How It Works

Customer Journey:

  1. Landing Page: Customer fills out contact form
  2. Contact Created: Automatically added to GHL with tags
  3. Photo Upload: Customer uploads their smile photo
  4. AI Transformation: Google Gemini generates enhanced smile
  5. Video Generation: Animated video created (optional)
  6. Data Synced: All images & videos saved to GHL contact
  7. Automation: Your workflows trigger for follow-up

Data Saved to Each Contact:

  • Full Name (split into first/last)
  • Email Address
  • Phone Number
  • Service Interest (Veneers, Invisalign, Whitening, etc.)
  • Notes/Concerns
  • Before/After Image URLs
  • Video URL (if generated)
  • Tags: "smile-transformation", "ai-lead"

🔧 Troubleshooting

Issue: Contacts Not Creating

Solution:

  • Verify API Key is saved in Settings → Integration
  • Check Location ID is correct
  • Ensure API key has contacts.write permission
  • Check browser console for errors

Issue: Videos Not Saving

Solution:

  • Verify smile_video_url custom field exists in GHL
  • Field type should be "Text" or "URL"
  • Check contact notes for the video link as backup

Issue: App Not Detecting GHL

Solution:

  • Verify you installed via GHL Marketplace
  • Check URL contains location_id parameter
  • Try manual configuration in Settings
